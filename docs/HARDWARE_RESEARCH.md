## Setup Requirements

### For Kitchen Wall Mount:

1. **Power:**
   - Need outlet near mount location
   - Or use long USB-C cable
   - Consider always-plugged-in setup

2. **WiFi:**
   - Strong WiFi signal needed
   - Tablet will be online-only (with database)
   - Consider WiFi extender if needed

3. **Location:**
   - Eye level for easy viewing
   - Protected from splashes/spills
   - Easy to reach for interaction

## PWA Installation on Android Tablets

### Steps:
1. Open Chrome browser
2. Navigate to your app URL
3. Tap menu (3 dots) → "Install app"
4. Confirm installation
5. App icon appears on home screen
6. Opens in standalone mode (no browser UI)
7. Works like native app

**Result:** Full PWA experience, no workarounds needed!
